## 에이전트 종류

### 정보 에이전트

- BB 밴드

### 리뷰 에이전트

실패한 매매에 대한 피드백을 진행합니다. 2일뒤에 피드백을 진행하여 아래 내용에 대해서 리뷰합니다.

- 시나리오가 자기가 예상한 방향으로 진행이 되었는지
- 틀렸으면 왜 틀렸는지 기재한다 예측을 못한 예외케이스의 경우 따로 기재한다.

### 시나리오 에이전트

차트의 패턴을 파악하여 가능한 시나리오를 작성한다.

아래 내용을 먼저 파악한다.

- 저항선은 어떻게 되는지
- 지지선은 어떻게 되는지

- 현재 차트 패턴은 어떻게 되는지

### 판단 에이전트

모든 정보,리뷰,시나리오를 총합하여 TP,SL를 정한 다음 해당 TP,SL를 정한다.
포지션의 최대 할당 가능 수준을 환경변수로 세팅한다.
레버리지의 경우 LLM에게 맡긴다.
이때 SL의 최대 허용 손실율은 환경변수로 세팅한다.
