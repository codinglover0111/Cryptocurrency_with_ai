## 에이전트 종류

### 정보 에이전트

아래 정보를 가져옵니다.

- OLCHV
- BB 밴드

### 리뷰 에이전트

실패한 매매에 대한 피드백을 진행합니다. 2일뒤에 피드백을 진행하여 아래 내용에 대해서 리뷰합니다.

- 시나리오가 자기가 예상한 방향으로 진행이 되었는지
- 불필요한 손절을 하였는지
- 익절 할 기회가 있었지만 이를 잘 해냈는지
- 시나리오가 틀렸을 경우 이에 대해서 잘 대응을 했는지

### 시나리오 에이전트

차트의 패턴을 파악하여 가능한 시나리오를 작성한다.

아래 내용을 먼저 파악한다.

- 저항선은 어떻게 되는지
- 지지선은 어떻게 되는지

- 현재 차트 패턴은 어떻게 되는지

### 판단 에이전트

모든 정보,리뷰,시나리오를 총합하여 진입가,TP,SL를 정한 다음 해당 TP,SL를 정한다.
자기가 지정한 시간 동안 유효한 1회 가격 알람을 띄운다.
포지션의 최대 할당 가능 수준을 환경변수로 세팅한다.
레버리지의 경우 LLM에게 맡긴다.
이때 SL의 최대 허용 손실율은 환경변수로 세팅한다.
